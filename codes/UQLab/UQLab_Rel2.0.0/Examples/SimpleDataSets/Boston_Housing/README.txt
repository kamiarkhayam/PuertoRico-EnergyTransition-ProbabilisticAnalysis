DATASET DESCRIPTION:
--------------------
 
Origin/references:
--------------------------
  Origin:
    The Boston housing dataset originates from Harrison et. al 1978 and is 
    publicly available at the UCI machine learning repository: https://archive.ics.uci.edu/ml/   
  
  References:
    Harrison, D. and Rubinfeld, D.L. (1978). Hedonic prices and the demand for clean air, J. Environ. 
    Economics & Management, vol.5, 81-102.
 
Dataset description:
--------------------------
  The dataset contains information collected by the U.S. Census Service concerning housing in the 
  area of Boston Massachusetts. It has been used extensively in the literature to benchmark learning
  algorithms. 
 
  Each input sample  consists of 14 variables, each related to specific metrics for a specific town. 
  The output of interest corresponds to the median value of owner-occupied homes in $1000's of that town.  
 
Description of inputs and outputs (ordered as provided in the dataset)
---------------------------
  housing.mat:
  X             : A set of N=506 samples of 1 binary and 12 real-valued input variables
  Y             : The corresponding samples of the output of interest
